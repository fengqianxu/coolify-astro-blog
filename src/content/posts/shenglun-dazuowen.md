> 核心理念：**审题是根，根基不稳，地动山摇。** 标题不是装饰，它等于总论点，也等于开头最后那句话。

---

## 第一步：审题——锁定题干关键词

大作文最怕的就是跑题。审题的核心动作只有一个：**找出题干中的关键词**，然后按照关键词数量，判断是哪类主题。

- **单主题**：题干只有一个核心概念
- **双主题**：题干有两个并列或关联的核心概念
- **多主题**：题干有三个及以上核心概念

> **注意**：为了不偏题，标题里要**原封不动地把关键词写上去**，尽量不要自己发挥替换词汇。

---

## 第二步：拟标题——万能公式

标题 = 总论点 = 开头最后一句话，三者是同一个东西。

### 核心公式：**对策 + 影响**

最常规的写法。对策在前，影响在后，中间用空格分开（不用逗号）。

**对策的万能写法**：用「以……」句式

| 模板 | 例子 |
|---|---|
| 以……破题 答好……答卷 | 以"一键办"破题 答好为民答卷 |
| 以……破局 促……行远 | 以反向思维破局 促正向发展行远 |
| 以……破局 谋……之路 | 以"通路"破局 谋发展"同路" |
| 以……之光 照……之路 | 以科技之光 照公益之路 |
| 以……为笔 绘……长卷 | 以创新之笔绘就广东新质生产力发展长卷 |
| 以……为笔 绘……新篇 | 以产业振兴破题 绘乡村振兴新篇 |
| 以……为笔 绘……长卷 | 以生产性服务业为笔 绘新质生产力发展长卷 |
| 以…… + 影响（无固定后缀）| 以良性互动开创善治新局面 |

---

### 影响的三种写法

**① 主体分析**：对个体 → 对组织 → 对国家/政府/社会

**② 关键词升华**：把题干关键词往更大的价值上引

**③ 万能模板句**：

- 推动高质量发展
- 增进民生福祉
- 开创发展新局面
- 共创美好未来
- 谱写发展新篇章
- 打造发展新引擎
- 压舱发展航船
- 构建和谐社会
- 成就人生价值
- 绽放青春风采
- 实现良法善治

---

## 单主题标题拟定

直接套用「对策 + 影响」公式，简单直接。以……句式的例子见上方模板表格。

## 双主题标题拟定

双主题分两类，处理方式不同。

### 类型一：AB 型（两个主题直接关联）

两个关键词可能是并列、对立、互补关系。有两种打法：

**打法一：当 A 与 B 互相影响、相辅相成时** → 用「双向奔赴」模板

```
A与B：双向奔赴 共创未来
A与B：相互融合 共创未来
```

| 例子 |
|---|
| 创新与风险：双向奔赴 共创未来 |
| 理论与实践：双向奔赴 共创未来 |
| 农民增收与农村提质消费：相互融合 双向奔赴 |
| 现代让传统鲜活 传统让现代厚重 |
| 传统现代 双向奔赴 |

**打法二：当 A、B 可分别作为对策时** → 直接用「对策 + 影响」

| 例子 | 备注 |
|---|---|
| 以民生"小事" 成国家"大事" | 对策是微观的，影响是宏观的 |
| 战胜困难与挑战 把握机遇与希望 | 将问题改编成了对策 |
| 肩负时代使命 赢得斗争胜利 | |
| 坚持群众路线 筑就美好生活 | |
| 数字时代需要公平竞争 | |

---

### 类型二：AB + C 型（两个对策 + 一个影响）

题干表面是两个词，但材料里藏着一个被这两个词共同指向的目标 C。格式：

```
A（对策一）+ B（对策二）+ C（影响/目标）
```

> **判断技巧**：看到 AB 型题干，别急着套模板，先看材料。如果材料里有一个两者都在服务的共同目标，那就是 AB+C；如果两者是互相影响的平行关系，就是纯 AB 型。

**万能模板**：

```
以……为笔，……为墨，擘画……新蓝图
以……为炬，……为火，照亮……之路
以……为根，……为叶，……之树繁茂永存
```

| 例子 |
|---|
| 智慧赋能 创新驱动 为行政执法提质增效 |
| 做实"里子" 扮靓"面子" 提升城市治理水平 |
| 以变与不变之笔 绘文化发展长卷 |
| 软硬兼施 提升城市治理精细化水平 |
| 传统为根 现代为叶 文明之树繁茂永存 |
| 兼顾有形与无形 打通文明发展道路 |
| 思维与播种实现高质量发展 |

---

## 多主题标题拟定

题干里有三个及以上的关键词。判断标准只有一个：**这几个词能不能合并成一个上位概念？**

### 能合并 → 提取共性对策 + 影响

三个词如果都属于同一类行为，可以用一个上位词概括，再加影响。

| 例子 | 备注 |
|---|---|
| 以"力""理""利"提升行政执法水平 | 三词直接入标题（同时也是共性对策）|
| 健全治理体系 实现乡村振兴 | "治理"概括了题干的法治、德治、自治 |

---

### 不好合并 → 三词直接并列

三个词用空格或间隔号（·）排列，视角可以都是对策，也可以都是影响。

| 例子 |
|---|
| 追求·努力·创新 |
| 心路·辛路·新路 |
| 新问题 新契机 新发展 |
| 始于梦想 基于创新 成于实干 |
| 守本分 强本领 勇创新 |
| 办好事 好办事 事办好 |

---

## 重要注意事项

### 1. 标题格式
- **不用逗号**，用空格分隔各部分
- 并列词语之间可用间隔号（·）

### 2. 关键词未完全入标题怎么办？

如果标题里没办法把题干所有关键词都写进去，那**开头第一段和结尾最后一段必须明确点出这些词**。目的只有一个：让阅卷老师看到你没有偏题。

## 一张图总结

```
题干关键词数量
│
├── 单主题 → 对策 + 影响（万能句式：以……）
│
├── 双主题
│   ├── AB 型（平行/互补）
│   │   ├── 互相影响 → A与B：双向奔赴 共创未来
│   │   └── 各自为策 → 对策A + 对策B（影响）
│   └── AB+C 型（两个手段+一个共同目标）→ A + B + C
│
└── 多主题
    ├── 能合并 → 共性对策 + 影响
    └── 不好合并 → 词一 词二 词三（并列）
```

## 开头写法

### 通用结构

开头的核心是：**一段话把「背景→问题→立场」交代清楚，最后一句收到总论点**。

**通用模板**：

```
名言警句 → 背景句（可选）→ 影响 → 成绩（可选）→ 问题（可选）→ 过渡句 → 总论点
```

**适用范围**：单主题、多主题、AB+C 型双主题、对策+影响类 AB 型双主题。

---

**三个注意点**：

**① 没有名言警句时，以背景句开头。** 不是每道题都能从题干或材料里找到合适的引用，强行凑不如直接写背景句。

**② 名言优先级：题干 → 材料 → 自己积累。** 顺序不是偶然的——越贴近题干，越不容易跑偏。自己积累的名言是最后备选，不是优先项。

**③ 总论点点题时，隐晦的标题需要解释。** 如果标题是「心路·辛路·新路」这类，单看标题不知道在说什么，那总论点这句话就得顺带说清楚：「为民服务的路，是"心路"，是"辛路"，也是"新路"，这意味着要听取民意、不怕困难、善于创新。」

---

### 辩证 AB 型——不能用通用模板

辩证 AB 型指的是：两个主题表面上看起来对立或割裂，但实则互相依存、相辅相成。比如「传统与现代」「创新与风险」「变与不变」。

这类题**不能用通用模板**。原因很简单：通用模板是线性推进的，套进去只能把 A 和 B 各说一遍，但写不出「两者有关联、不应被对立」这层意思——而这恰恰是这类题的核心立场。

以下是三个可用的基础模板，可以放在开头，也可以作为从开头到分论点的过渡段。后期可以把 A、B 换成具体词，再打磨语感，改成自己的风格。

---

**模板一（关系型）**：先肯定进步，再点出 A 与 B 不冲突，最后说明两者如何相互成就。

> "历尽天华成此景，人间万事出艰辛"，近些年来，我们在统筹把握 A 与 B 方面取得了长足的进步。能否统筹把握好 A 和 B，事关人民群众的根本利益，事关政府治理能力和治理水平的提升，事关社会主义现代化国家的建设。进入新时代，奋进新征程，再创新辉煌，我们应该认识到 A 与 B 并不冲突，二者相互统一，**A 让 B 更亲切鲜活，B 让 A 更丰富厚重**，A 与 B 相互融合让中华文明永远焕发新机。

---

**模板二（破立型）**：先制造「A 与 B 是否对立」的悬念，再用核心句破除对立，得出两者相辅相成的结论。

> 在当今社会快速发展的浪潮中，A 与 B 似乎成了两个截然不同的概念。但是，A 与 B 是对立关系还是统一关系，是只能孤立割裂还是相互融合促进？"**A 让 B 更亲切鲜活，B 让 A 更丰富厚重。**"这句话为我们提供了一个独特的视角来看待 A 与 B 之间的相互影响和共生关系，即我们应该认识到 A 与 B 虽然有区别，但两者并不冲突，而是相辅相成、互相统一的关系，二者相得益彰，双向奔赴。

---

**模板三（情境型）**：先用一两句话描绘 A 与 B 各自的具体形象，制造画面感，再引出核心句破除对立。比模板二更有现场感，适合主题本身形象鲜明的题目（如传统与现代）。

> 在现代社会中，我们身处于瞬息万变的科技发展与传统文化之间。传统是一种情感的寄托，是历史的痕迹，是民族的记忆；而现代则是科技的力量，是时间的脉络，是创新的源泉。"**A 让 B 更亲切鲜活，B 让 A 更丰富厚重。**"这句话为我们提供了一个独特的视角来看待 A 与 B 之间的相互影响和共生关系，即我们应该认识到 A 与 B 虽然有区别，但两者并不冲突，而是相辅相成、相互统一的关系，二者相得益彰，双向奔赴。

> **用法提示**：三个模板里的核心句「A 让 B 更亲切鲜活，B 让 A 更丰富厚重」都可以替换，换成更贴合当前题目关系的说法效果更好。模板一适合开门见山、正向铺陈；模板二适合先抛问题、再反转破局；模板三适合主题形象鲜明、想先渲染画面再破题的情况。
