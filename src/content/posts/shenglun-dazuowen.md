> 核心理念：**审题是根，根基不稳，地动山摇。** 标题不是装饰，它等于总论点，也等于开头最后那句话。

---

## 第一步：审题——锁定题干关键词

大作文最怕的就是跑题。审题的核心动作只有一个：**找出题干中的关键词**，然后按照关键词数量，判断是哪类主题。

- **单主题**：题干只有一个核心概念
- **双主题**：题干有两个彼此独立、地位平等的核心概念
- **多主题**：题干有三个及以上核心概念

> **特别注意**：题干出现两个词，不一定就是双主题。如果这两个词是「一个是手段/方法，另一个是目标/结果」的关系，本质上是**单主题**，按单主题处理。例如「流动与新生」——流动是对策，新生是影响——拟标题时用「对策 + 影响」公式，不要套双主题模板。

> **不偏题原则**：标题里要**原封不动地把关键词写上去**，尽量不要自己发挥替换词汇。

---

## 第二步：拟标题——万能公式

标题 = 总论点 = 开头最后一句话，三者是同一个东西。

### 核心公式：**对策 + 影响**

最常规的写法。对策在前，影响在后，中间用空格分开（不用逗号）。

**对策的万能写法**：用「以……」句式

| 模板 | 例子 |
|---|---|
| 以……破题 答好……答卷 | 以"一键办"破题 答好为民答卷 |
| 以……破局 促……行远 | 以反向思维破局 促正向发展行远 |
| 以……破局 谋……之路 | 以"通路"破局 谋发展"同路" |
| 以……之光 照……之路 | 以科技之光 照公益之路 |
| 以……为笔 绘……长卷 | 以创新之笔绘就广东新质生产力发展长卷 |
| 以……为笔 绘……新篇 | 以产业振兴破题 绘乡村振兴新篇 |
| 以……为笔 绘……长卷 | 以生产性服务业为笔 绘新质生产力发展长卷 |
| 以…… + 影响（无固定后缀）| 以良性互动开创善治新局面 |

---

### 影响的三种写法

**① 主体分析**：对个体 → 对组织 → 对国家/政府/社会

**② 关键词升华**：把题干关键词往更大的价值上引

**③ 万能模板句**：

- 推动高质量发展
- 增进民生福祉
- 开创发展新局面
- 共创美好未来
- 谱写发展新篇章
- 打造发展新引擎
- 压舱发展航船
- 构建和谐社会
- 成就人生价值
- 绽放青春风采
- 实现良法善治

---

## 单主题标题拟定

直接套用「对策 + 影响」公式，简单直接。以……句式的例子见上方模板表格。

---

## 双主题标题拟定

双主题分两类，处理方式不同。

### 类型一：AB 型（两个主题互相关联）

两个关键词彼此独立但相互影响，可能是并列、对立、互补的关系。

> **先做一步判断**：如果题干的两个词其实是「一个是手段、另一个是结果」，那它是**单主题**，按上面单主题处理，不要套这里的模板。只有两个词地位平等、互相影响时，才是真正的 AB 型。

**万能模板**：

```
A与B：双向奔赴 共创未来
A与B：相互融合 共创未来
```

| 例子 |
|---|
| 创新与风险：双向奔赴 共创未来 |
| 理论与实践：双向奔赴 共创未来 |
| 农民增收与农村提质消费：相互融合 双向奔赴 |
| 现代让传统鲜活 传统让现代厚重 |
| 传统现代 双向奔赴 |

---

### 类型二：AB + C 型（两个对策 + 一个影响）

题干表面是两个词，但材料里藏着一个被这两个词共同指向的目标 C。格式：

```
A（对策一）+ B（对策二）+ C（影响/目标）
```

> **判断技巧**：看到 AB 型题干，别急着套模板，先看材料。如果材料里有一个两者都在服务的共同目标，那就是 AB+C；如果两者是互相影响的平行关系，就是纯 AB 型。

**万能模板**：

```
以……为笔，……为墨，擘画……新蓝图
以……为炬，……为火，照亮……之路
以……为根，……为叶，……之树繁茂永存
```

| 例子 |
|---|
| 智慧赋能 创新驱动 为行政执法提质增效 |
| 做实"里子" 扮靓"面子" 提升城市治理水平 |
| 以变与不变之笔 绘文化发展长卷 |
| 软硬兼施 提升城市治理精细化水平 |
| 传统为根 现代为叶 文明之树繁茂永存 |
| 兼顾有形与无形 打通文明发展道路 |
| 思维与播种实现高质量发展 |

---

## 多主题标题拟定

题干里有三个及以上的关键词。判断标准只有一个：**这几个词能不能合并成一个上位概念？**

### 能合并 → 提取共性对策 + 影响

三个词如果都属于同一类行为，可以用一个上位词概括，再加影响。

| 例子 | 备注 |
|---|---|
| 以"力""理""利"提升行政执法水平 | 三词直接入标题（同时也是共性对策）|
| 健全治理体系 实现乡村振兴 | "治理"概括了题干的法治、德治、自治 |

---

### 不好合并 → 三词直接并列

三个词用空格或间隔号（·）排列，视角可以都是对策，也可以都是影响。

| 例子 |
|---|
| 追求·努力·创新 |
| 心路·辛路·新路 |
| 新问题 新契机 新发展 |
| 始于梦想 基于创新 成于实干 |
| 守本分 强本领 勇创新 |
| 办好事 好办事 事办好 |

---

## 重要注意事项

### 1. 标题格式
- **不用逗号**，用空格分隔各部分
- 并列词语之间可用间隔号（·）

### 2. 关键词未完全入标题怎么办？

如果标题里没办法把题干所有关键词都写进去，那**开头第一段和结尾最后一段必须明确点出这些词**。目的只有一个：让阅卷老师看到你没有偏题。

---

## 一张图总结

```
题干关键词数量
│
├── 单主题 → 对策 + 影响（万能句式：以……）
│
├── 双主题
│   ├── AB 型（平行/互补）→ A与B：双向奔赴 共创未来
│   └── AB+C 型（两个手段+一个共同目标）→ A + B + C
│
└── 多主题
    ├── 能合并 → 共性对策 + 影响
    └── 不好合并 → 词一 词二 词三（并列）
```

## 开头写法

### 通用结构

开头的核心是：**一段话把「背景→问题→立场」交代清楚，最后一句收到总论点**。

**通用模板**：

```
名言警句 → 背景句（可选）→ 影响 → 成绩（可选）→ 问题（可选）→ 过渡句 → 总论点
```

**适用范围**：单主题、多主题、AB+C 型双主题。

---

**三个注意点**：

**① 没有名言警句时，以背景句开头。** 不是每道题都能从题干或材料里找到合适的引用，强行凑不如直接写背景句。

**② 名言优先级：题干 → 材料 → 自己积累。** 顺序不是偶然的——越贴近题干，越不容易跑偏。自己积累的名言是最后备选，不是优先项。

**③ 总论点点题时，隐晦的标题需要解释。** 如果标题是「心路·辛路·新路」这类，单看标题不知道在说什么，那总论点这句话就得顺带说清楚：「为民服务的路，是"心路"，是"辛路"，也是"新路"，这意味着要听取民意、不怕困难、善于创新。」

---

### 辩证 AB 型——不能用通用模板

辩证 AB 型指的是：两个主题表面上看起来对立或割裂，但实则互相依存、相辅相成。比如「传统与现代」「创新与风险」「变与不变」。

这类题**不能用通用模板**。原因很简单：通用模板是线性推进的，套进去只能把 A 和 B 各说一遍，但写不出「两者有关联、不应被对立」这层意思——而这恰恰是这类题的核心立场。

以下是三个可用的基础模板，可以放在开头，也可以作为从开头到分论点的过渡段。后期可以把 A、B 换成具体词，再打磨语感，改成自己的风格。

---

**模板一（关系型）**：先肯定进步，再点出 A 与 B 不冲突，最后说明两者如何相互成就。

> "历尽天华成此景，人间万事出艰辛"，近些年来，我们在统筹把握 A 与 B 方面取得了长足的进步。能否统筹把握好 A 和 B，事关人民群众的根本利益，事关政府治理能力和治理水平的提升，事关社会主义现代化国家的建设。进入新时代，奋进新征程，再创新辉煌，我们应该认识到 A 与 B 并不冲突，二者相互统一，**A 让 B 更亲切鲜活，B 让 A 更丰富厚重**，A 与 B 相互融合让中华文明永远焕发新机。

---

**模板二（破立型）**：先制造「A 与 B 是否对立」的悬念，再用核心句破除对立，得出两者相辅相成的结论。

> 在当今社会快速发展的浪潮中，A 与 B 似乎成了两个截然不同的概念。但是，A 与 B 是对立关系还是统一关系，是只能孤立割裂还是相互融合促进？"**A 让 B 更亲切鲜活，B 让 A 更丰富厚重。**"这句话为我们提供了一个独特的视角来看待 A 与 B 之间的相互影响和共生关系，即我们应该认识到 A 与 B 虽然有区别，但两者并不冲突，而是相辅相成、互相统一的关系，二者相得益彰，双向奔赴。

---

**模板三（情境型）**：先用一两句话描绘 A 与 B 各自的具体形象，制造画面感，再引出核心句破除对立。比模板二更有现场感，适合主题本身形象鲜明的题目（如传统与现代）。

> 在现代社会中，我们身处于瞬息万变的科技发展与传统文化之间。传统是一种情感的寄托，是历史的痕迹，是民族的记忆；而现代则是科技的力量，是时间的脉络，是创新的源泉。"**A 让 B 更亲切鲜活，B 让 A 更丰富厚重。**"这句话为我们提供了一个独特的视角来看待 A 与 B 之间的相互影响和共生关系，即我们应该认识到 A 与 B 虽然有区别，但两者并不冲突，而是相辅相成、相互统一的关系，二者相得益彰，双向奔赴。

> **用法提示**：三个模板里的核心句「A 让 B 更亲切鲜活，B 让 A 更丰富厚重」都可以替换，换成更贴合当前题目关系的说法效果更好。模板一适合开门见山、正向铺陈；模板二适合先抛问题、再反转破局；模板三适合主题形象鲜明、想先渲染画面再破题的情况。

## 寻找分论点

### 第一步：确定文体角度

写分论点之前，先看题目要求的是哪种文体。文体决定分论点的结构公式：

| 文体 | 分论点结构 |
|---|---|
| 策论文 | 总论点 + 对策 |
| 政论文 | 总论点 + 影响 |
| 议论文 | 总论点 + 对策 + 影响 |

---

### 第二步：确定来源

分论点有三个来源，效果从高到低：

**题干 > 材料 > 小题**

题干最贴题，优先从题干里找；材料次之，内容直接来自给定信息；参考小题时，要注意对齐小题题干的表述角度，分论点要和小题保持一致。

---

### 第三步：从材料中找对策

在材料里找对策，可以从三个角度切入：

- **谁来干**（who）：主体是谁，谁在负责
- **怎样干**（how）：用什么方式、手段、机制
- **干哪方面**（what）：具体做的是哪个事情

对策在材料里的表现形式通常是：**动词 + 事情**。看到这种结构就要留意，它很可能就是一个分论点的核心。

---

### 如何快速读材料

不是每一句话都要仔细看，找分论点时有几个提效技巧：

- **每段首尾句**：先看这两句，判断这段话的谈话范围
- **高频词 / 关键词**：反复出现的词就是这段的核心话题
- **跳过故事性内容**：叙事、举例的段落先略过，重点看规范性表述（直接说「要做什么」「应该如何」的部分）
- **总结句优先**：材料里的总结归纳句往往直接就是分论点的雏形

---

### 按主题类型拆分论点

#### 单主题

分论点直接从材料或题干里找，按文体角度提炼即可（对策 / 影响 / 对策+影响）。

#### AB 型双主题

AB 型有两种拆法，选哪种取决于题干传递的信息：

**拆法一（互相影响型）**：A 与 B 是平等的两方，彼此影响
- 分论点一：A 对 B 的影响
- 分论点二：B 对 A 的影响
- 分论点三：A + B 共同的对策

**拆法二（正反影响型）**：题干暗示 A 对 B 同时有好有坏两面影响
- 分论点一：A 对 B 的正面影响
- 分论点二：A 对 B 的负面影响
- 分论点三：A + B 的对策

<!-- 例子待补充 -->

#### AB + C 型双主题

A 和 B 是两个手段，C 是共同目标。分论点有两种拆向：

**正向（从 A、B 出发指向 C）**：
- 分论点一：A → C
- 分论点二：B → C
- 分论点三：A + B → C

**反向（从 C 出发拆解 A、B）**：
- 分论点一：C 需要 A
- 分论点二：C 需要 B
- 分论点三：C 需要 A + B

<!-- 例子待补充 -->

#### 多主题

A、B、C 是三个并列主题，D 是提取出来的共性目标。拆法同样有两个方向：

**正向**：
- 分论点一：A → D
- 分论点二：B → D
- 分论点三：C → D

**反向**：
- 分论点一：D 需要 A
- 分论点二：D 需要 B
- 分论点三：D 需要 C

### 写分论点时的注意事项

**① 摘抄要完整，不漏词。** 从材料里提炼出来的内容，尽量用原文原词，不要自行改写。改词容易改偏，原词才紧扣材料。

**② 分论点之间必须并列。** 几个分论点要从同一个角度展开——要么都是对策，要么都是影响，要么都是主体——不能混用角度。

**③ 遇到模糊词必须解释。** 分论点中如果出现含义不明确的词，必须在后面跟一句解释，否则阅卷老师一头雾水，整个论点失效。例如题干是「心路·辛路·新路」，分论点不能只写「走好心路」，要接一句「所谓"心路"，就是……」。

## 论述段写法

### 标准模板

每个分论点下的论述段，遵循统一结构：

```
分论点 → 名言警句 → 背景 → 影响 → 问题 → 对策 → 回扣分论点
```

一句话记忆法：**「中好坏策」**——中性陈述（背景）、好的一面（影响）、坏的一面（问题）、怎么办（对策）。这四个维度可以组合穿插使用，不一定四个都写，根据材料给的信息取舍。

回扣分论点是结尾必做的动作，通常用「对策 + 影响」句式把分论点再点一次，让论证闭环。

---

### 三种论证方式

| 方式 | 内容构成 | 主要来源 |
|---|---|---|
| 要素论证 | 背景、问题、影响、原因、对策等要素堆叠 | 改写材料 / 自身积累 |
| 事例论证 | 举例子：问题、主体、对策、影响 | 改写材料 / 自身积累 |
| 名言警句论证 | 用名言直接佐证 | 自身积累 |

> 要素论证和事例论证都可以来源于**客观小题**的材料，小题已经帮你筛过一次核心信息，直接改写即用。

---

### 怎么选论证方式？看材料给你什么

论证方式不是随便挑的，本质上由**材料的组织形式**决定：

- **材料是「问题 + 对策」的规范性陈述** → 用**要素论证**，把材料里的要素（问题、原因、对策）抽出来改写
- **材料是故事、案例、人物事迹** → 用**事例论证**，把故事拆成「问题—主体—对策—影响」四元素再复述
- **缺少材料支撑、或需要拔高立意时** → 补一句**名言警句论证**

---

### 什么能背，什么不能背

备考中会接触大量积累资料，要分清哪些能背、哪些必须来自材料：

**可以提前背**

- 名言警句
- 影响（"三个事关"句式、万能模板句）

**必须来自材料**

- 背景
- 问题
- 对策

**为什么这么分？** 名言和影响是通用的——同一句"历尽天华成此景"可以用在任何主题上，同一组"三个事关"也只是主语变一下。但**背景、问题、对策一旦脱离材料，就没法证明你是在论证给定的题**，相当于把"是否扣题"这件事交给阅卷人去校验，风险太高。

所以，**背来的问题/对策只能是兜底方案**，正文里这类内容的比例应尽量压低。

### 不过，积累本身仍然值得

虽然背的材料不适合直接往正文塞，但整个积累过程并不是浪费：

- **阅历扩展**：大量接触政论文本，对题干敏感度和立意高度都有帮助
- **面试有用**：公考笔试之后还有面试，结构化答题时，积累下来的背景、案例、金句都能派上用场
- **应急兜底**：如果某道题材料特别薄，背来的内容就是救命稻草

积累和扣材料不冲突——**扣材料是主策略，背资料是备胎**。
